# EU Calm — Rust Server Plugins

Custom Oxide plugins and website for **EU Calm**, a chill Rust server at [calmservers.com](https://calmservers.com).

---

## Plugins

| Plugin | Description |
|---|---|
| **CalmLoot** | Custom loot tables for all containers — barrels, crates, heli, bradley, supply drop and more |
| **CalmProtection** | Offline raid protection with scaling damage reduction based on time offline |
| **CalmNight** | Vote-skippable nights without breaking server events |
| **CalmVote** | In-game voting system |
| **CalmHelper** | Utility helpers for server management |
| **CalmAnnouncer** | Automated server announcements |

## Website

Static site hosted at [calmservers.com](https://calmservers.com), including a plugin changelog page at `/updates.html`.
